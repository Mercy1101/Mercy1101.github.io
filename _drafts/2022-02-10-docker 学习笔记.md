# docker 学习笔记

## 拉取镜像到本地

```text
docker pull ubuntu
```

## 查询镜像列表

```text
docker images
```

## 删除镜像

```text
docker rmi <image id>
```

## 查看运行中的容器

```text
docker ps
```

## 查看所有容器

```text
docker ps -a
```

## 删除容器

```text
docker rm <container id>
```

## 从镜像创建一个容器

```text
docker create -p 3000:80 --name exampleApp3000 yoyomooc/exampleapp
```

`-p 3000:80`: 映射容器内端口 `80` 到主机端口 `3000`
`--name`: 为容器指定一个名字
`yoyomooc/exampleapp`: 指定使用该镜像创建容器
`-i`: Keep STDIN open even if not attached

## 停止一个容器

```text
docker stop <container ID>
```

## 启动容器

```text
docker start <container id / container name>
```

## 启动所有容器

```text
docker start $(docker ps -aq)
```

## 停止所有容器

```text
docker stop $(docker ps -q)
```

## bat 清理 docker 操作

windows 下脚本

```text
@echo off
:: 停止所有容器
FOR /f "tokens=*" %%i IN ('docker ps -q') DO docker stop %%i
:: 删除所有容器
FOR /f "tokens=*" %%i IN ('docker ps -aq') DO docker rm %%i
:: 删除所有镜像
FOR /f "tokens=*" %%i IN ('docker images --format "{{.ID}}"') DO docker rmi %%i
```

## 映射端口

### 将容器暴露的所有端口，都随机映射到宿主机上(不推荐)

```text
docker run -P -it ubuntu /bin/bash
```

### 将容器指定端口随机映射到宿主机一个端口上

```text
docker run -P 80 -it ubuntu /bin/bash
```

### 将容器指定端口指定映射到宿主机的一个端口上

容器内`80`端口映射到宿主机`8000`端口

```text
docker run -p 8000:80 -it ubuntu /bin/bash
```

### 将容器ip和端口，指定映射到宿主机上

将容器的ip192.168.0.100和80端口，映射到宿主机的8000端口。

```text
docker run -p 192.168.0.100:8000:80 -it ubuntu /bin/bash
```

### 查看映射端口配置

```text
docker port container_ID #容器ID
```
