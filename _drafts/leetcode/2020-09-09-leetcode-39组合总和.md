## 39. Combination Sum

Given a set of candidate numbers (candidates) (without duplicates) and a target number (target), find all unique combinations in candidates where the candidate numbers sums to target.

The same repeated number may be chosen from candidates unlimited number of times.

Note:

All numbers (including target) will be positive integers.
The solution set must not contain duplicate combinations.

Example 1:

Input: candidates = [2,3,6,7], target = 7,
A solution set is:
[
  [7],
  [2,2,3]
]

Example 2:

Input: candidates = [2,3,5], target = 8,
A solution set is:
[
  [2,2,2,2],
  [2,3,3],
  [3,5]
]
 

Constraints:

* 1 <= candidates.length <= 30
* 1 <= candidates[i] <= 200
* Each element of candidate is unique.
* 1 <= target <= 500


```c++
class Solution {
 public:
  vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
    std::sort(candidates.begin(), candidates.end());
    std::vector<std::vector<int>> res;
    std::pair<int, std::vector<int>> temp;
    temp.first = 0;
    std::vector<int> sec;
    temp.second = sec;
    std::function<void(size_t index)> sum_func = [&](size_t index) {
      if (index >= candidates.size()) {
        return;
      }

      if (temp.first > target) {
        /// 注意这里不能pop应该直接return，否则会死循环
        return;
      } else if (temp.first == target) {
        res.push_back(temp.second);
        return;
      } else {
        /// 跳过该数字
        sum_func(index + 1);
        /// 考虑当前数字
        temp.first += candidates.at(index);
        temp.second.push_back(candidates.at(index));
        sum_func(index);
        temp.first -= temp.second.back();
        temp.second.pop_back();
      }
    };
    sum_func(0);
    return res;
  }
};
```
