## 96. Unique Binary Search Trees

Given n, how many structurally unique BST's (binary search trees) that store values 1 ... n?

Example:

Input: 3

Output: 5

Explanation:

Given n = 3, there are a total of 5 unique BST's:

```txt
   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
```

 

Constraints:

1 <= n <= 19

```c++
class Solution {
public:
    int numTrees(int n) {
      std::vector<int> helper(n+1,0);
      helper.at(0) = 1;
      helper.at(1) = 1;
      for(int i = 2; i <= n; ++i){
        for(int j = 1; j <= i; ++j){
          helper.at(i) += helper.at(j-1) * helper.at(i-j);
        }
      }
      return helper.back();
    }
};
```

卡塔兰数
```c++
class Solution {
public:
    int numTrees(int n) {
        long long C = 1;
        for (int i = 0; i < n; ++i) {
            C = C * 2 * (2 * i + 1) / (i + 2);
        }
        return (int)C;
    }
};
```
